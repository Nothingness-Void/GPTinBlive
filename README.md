# GPTinBlive

一个将GPT接入哔哩哔哩直播的项目,目前实现了自动回复弹幕以及礼物并语音输出的功能

闲着没事自己做着玩的，之后不一定更不更新




### 运行环境

- Python 3.6+
- Windows操作系统

### 安装依赖

在命令行中使用以下命令安装所需库：

`pip install -r requirements.txt`

### 使用

1. 用代码编辑工具打开main.py文件。
2. 在# ChatGPT API的URL和密钥中设置api链接（这个可以不改）和你的APIKey
3. 按照您的意愿调整机器人的预设。
4. 用Python运行main.py文件。
5. 输入要连接的B站直播间编号。
6. 按下`Enter`键开始监听弹幕流。
